Viktiga CSS selektorer och egenskaper

Box-modellen

  1. margin
  2. padding
  3. border
  4. dessutom: outline (en linje utanför vanlig border), med outline-style, outline-color, outline-width, etc

Display

  1. block
  2. inline
  3. none - tar bort elementet (jmf. visibility: hidden - (inte display) som gör elementet osynligt, men det tar plats
  4. flex
  5. grid
  6. table
  7. etc.

Position

  1. static
  2. relative
  3. fixed
  4. absolute
  5. sticky

Float

  1. left
  2. right
  3. none
  4. inherit (ärver float från sin parent)
  5. .clearfix::after {
    content: ""; clear: both; display: table; }

z-index

  1. positivt värde (15)
  2. negativt värde (-6)
